# dwarf-c-reconstructor

```
DWARF C reconstructor - generates C++ code from DWARF debugging information

Usage: dwarf-c-reconstructor [OPTIONS] <ELF_FILE>

Arguments:
  <ELF_FILE>  Path to the ELF file to analyze

Options:
  -o, --output-dir <OUTPUT_DIR>  Output directory for generated files [default: output]
  -h, --help                     Print help
  -V, --version                  Print version
```
